#### 모듈 설치
* pip install requests
    > HTTP Request 옵션을 사용할 수 있다(GET, POST, DELETE ...)
* pip install bs4
    > 스프래핑에 사용, 흔히 크롤링 한다고 표현
* pip install playwright
	* cmd 새로 실행 후 playwright install 실행
		> 브라우저를 조작할 수 있음
* import time
    > 스레드 컨트롤
* import csv
    > csv 파일 생성, file = open('save path', 'mode') ... file.close()
